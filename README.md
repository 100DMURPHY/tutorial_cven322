# CVEN322 Civil Engineering Systems Teaching Platform

A public learning platform focused on Engineering Economics, Optimization, and Simulation.

## How to Run

You can run this application directly from GitHub using `uvx`:

```bash
uvx --from git+https://github.com/USER/REPO cven-app
```

## Features
- Interactive Engineering Economics calculators.
- Optimization model visualizers.
- Systems simulation tools.
