# CVEN322 Civil Engineering Systems

Welcome to the documentation for the CVEN322 Civil Engineering Systems teaching platform.

## Overview
This platform is designed to provide transparent, high-quality teaching materials for Civil Engineering systems, including Optimization, Engineering Economics, and Simulation.

## Features
- **Interactive Calculators**: Real-time analysis of engineering problems.
- **Visual Learning**: Advanced charting with Echarts and Plotly.
- **Easy Execution**: Run directly via `uvx`.

## Quick Start
To run the application locally:
```bash
uvx --from git+https://github.com/USER/REPO cven-app
```
